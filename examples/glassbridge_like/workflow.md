# Workflow

## Objective

Assess whether a hypothetical optical interposer can replace active FAU alignment.

## Evidence Grades

- `E1`: direct measured data.
- `E2`: official/vendor/conference claim.
- `E3`: same-class literature.
- `E4`: model assumption.
- `PENDING`: not yet supported.

## Same-Layer Rule

Compare interface schemes with interface schemes. Keep CPO platforms and roadmaps as system background unless they expose the actual optical interface and measurements.
