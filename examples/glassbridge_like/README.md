# GlassBridge-Like Example

This is a tiny example project showing how to use the research-audit template for a hardware/photonic-packaging feasibility question.

It is intentionally small. The purpose is to show file roles, not to reproduce the full GlassBridge report.

## Commands

```bash
python3 scripts/check_project.py
python3 scripts/build_graph_from_ledger.py
```
